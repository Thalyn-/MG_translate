How to create Monero wallet?

  1. Download the GUI WALLET (BETA)

  2. Download the Monero files depending of your OS : Windows 64-bit binaries (0.8.8.2, updated: May 21st) or Windows 32-bit binaries (0.8.8.2, updated: May 21st)
  3. Extract all the files and put them in the same folder.

  4. Open CryptNoteWallet.exe
    (Download & install the .NET Framework Version 4.5.2 if needed.)
  5. Choose your wallet name the first time you launch.
  6. You can use the GUI wallet as soon as the blockchain is downloaded.
